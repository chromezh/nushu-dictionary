简介

目前，女书字符已进入国际编码标准，可以在网页上以文字形式呈现。本指南的目标是指导网站制作者排版女书网页，对女书艺术品设计者也有参考价值。

lang 标签

对女书文本，HTML 的 lang 标签使用 lang="zh-Nshu"。这是因为女书记录的语言为江永土话，属于汉语方言的一种，故语言标签为 zh；女书的文字代码为 Nshu。

字体

目前可以显示女书字符的字体为 Unicode Nushu。它能正常显示 396 个女书字符和 1 个女书标点。

下載链接，或直接在 CSS 中引入：

  • TTF 格式：https://nushuscript.org/fonts/Uni1B170Nushu.ttf
  • WOFF 格式：https://nushuscript.org/fonts/Uni1B170Nushu.woff
  • WOFF2 格式：https://nushuscript.org/fonts/Uni1B170Nushu.woff2

该字体是从 Unicode V12.1 相关文件中直接提取，版权为字体制作者所有，在线女书字典网站并无该字体的版权，使用时请注意版权问题。

Unicode V12.1 相关文件：

  • https://unicode.org/charts/PDF/U1B170.pdf
  • https://unicode.org/charts/PDF/U16FE0.pdf

本网站原来使用 TH-Tshyn-P1 字体，但发现该字体存在一处错误，U+1B201 与 U+1B202 两个女书字形颠倒，因此不再推荐使用。
